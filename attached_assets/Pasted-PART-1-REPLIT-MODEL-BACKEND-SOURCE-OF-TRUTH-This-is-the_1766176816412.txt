🧠 Advanced AI Execution Logic (internal)

This is not exposed, but this is the rule that keeps you safe:

def should_run_advanced_ai(user, candidate, risk_score):
    plan = PLANS[user.plan]

    if not plan["advanced_ai_enabled"]:
        return False

    if candidate.advanced_ai_ran:
        return False

    if risk_score < 65:
        return False

    if user.advanced_ai_used_this_month >= plan["advanced_ai_monthly_budget"]:
        return False

    return True


If it returns False:
