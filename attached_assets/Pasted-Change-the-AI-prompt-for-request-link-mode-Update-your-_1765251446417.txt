Change the AI prompt for request_link mode

Update your Replit prompt so the model expects a real URL from your backend and uses it in the text:

Inputs:
- candidateName
- positionAppliedFor
- referenceSubmissionUrl  // <- actual link generated by the app

When mode = "request_link":
- Write a short, professional message the recruiter can send to the candidate.
- The message MUST include the exact value of `referenceSubmissionUrl` as the link to submit references.
- Explain that the link lets them provide:
  - Reference name
  - Reference email address
  - Relationship to candidate
  - Confirmation/consent to be contacted

Return JSON:

{
  "candidateLinkRequestText": "..."
}

2. Backend: generate the actual link

On your server, when they click Generate Link Request:

// pseudo-code
const refRequestId = await db.createReferenceRequest({
  candidateId,
  positionAppliedFor,
  status: "pending"
});

const token = createSignedToken(refRequestId); // JWT or random UUID
const referenceSubmissionUrl = `https://your-app.com/reference/${token}`;


Then call the AI with:

const aiInput = {
  mode: "request_link",
  candidateName,
  positionAppliedFor,
  referenceSubmissionUrl
};

3. Candidate-facing form (what the link goes to)

That URL should open a simple page:

Fields:

Candidate name (read-only or hidden from token)

Position (read-only)

Reference Name

Reference Email

Relationship to Candidate (dropdown)

“I confirm this person may be contacted as a reference” checkbox

Submit button

On submit, you save:

db.updateReferenceRequest(refRequestId, {
  referenceName,
  referenceEmail,
  relationship,
  consentGiven: true,
  status: "ready"
});

4. Updated UI copy for the right-hand panel

Replace the placeholder email with something like:

As part of our hiring process for the {{positionAppliedFor}} position, we’d like to gather your professional references.

Please use the link below to securely provide your reference details:
{{referenceSubmissionUrl}}

We will ask for:
- Reference name
- Reference email address
- Their relationship to you
- Confirmation that they agree to be contacted

Thank you for taking a moment to complete this step in the hiring process.

Best regards,
{{companyName}}
