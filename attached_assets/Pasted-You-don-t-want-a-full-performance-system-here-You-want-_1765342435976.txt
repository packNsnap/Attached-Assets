Returns all goals plus computed summary:

total_goals

completed_goals

at_risk_goals

POST /api/performance/goals/generate
Body:

{
  "employee_id": "123",
  "employee_name": "Ariella Harlin",
  "role": "Verification Specialist",
  "industry": "Background Screening",
  "time_horizon": "90 days"
}


Steps:

Build a search query like:
"SMART performance goals for Verification Specialist in background screening 90 days".

Call a web search API (using SEARCH_API_KEY) and get the top 3–5 results with title, url, snippet.

Call OpenAI (Chat Completions, e.g. gpt-4.1-mini) with:

System: you are an HR performance coach; create measurable SMART goals using the search snippets as context; do not claim legal or HR compliance guarantees.

User: include employee_name, role, industry, time_horizon, and the search results.

Ask OpenAI to return strict JSON:

{
  "goals": [
    {

Return the created goals.

PATCH /api/performance/goals/:id

status

due_date
